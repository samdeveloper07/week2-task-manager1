"# Week 2 - Task Manager" 

Week 2 Project — Task Manager Web Application
PROJECT DESCRIPTION
A responsive task manager web application built using HTML5, CSS3, and vanilla JavaScript. Allows users to create, manage, and track their daily tasks with a clean UI and persistent local storage support. Built by Shaik Sameer, a Java Full Stack Developer.
TECHNOLOGIES USED

HTML5 (semantic elements)
CSS3 (Flexbox, Grid, animations, media queries)
JavaScript (DOM manipulation, Local Storage API)
Multi-file JS architecture (app, ui, storage, utils)

FEATURES

Fully responsive design (mobile, tablet, desktop)
Add, edit, and delete tasks
Mark tasks as complete / incomplete
Theme switcher (Light and Dark mode)
Persistent data using Local Storage (tasks saved on refresh)
Clean and interactive UI with smooth animations
Modular JavaScript structure for maintainability

SETUP INSTRUCTIONS

Clone the repo:
git clone https://github.com/samdeveloper07/week2-task-manager1.git
Open the folder in VS Code
Open index.html in your browser
No build step or dependencies required
